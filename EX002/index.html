Nesta frase, tenho um termo em Negrito usando a tag STRONG (Forma e Sentindo no Negrito).

Texto Marcado

Podemos criar também um texto marcado usando a tag MARK.

Estamos criando o texto grande e um texto pequeno nesse parágrafo.

Texto Deletado

Podemos marcar um texto excluído para indicar que ele deve ser lido, mas não considerado. usando a tag DEL (semântica).

Texto Inserido

Podemos marcar um texto inserido (texto sublinhado) para dar uma ênfase e indicar que ele foi adicionado. usando a tag INS (semântica).

Para inserir coisas do tipo x20+3 usando a tag SUP

Aula 15

Colocando audio no seu site

Dicas de musicas sem direitos autorais: Youtube/Youtube studio/Biblioteca de audio/Atribuição não é necessária

Reproduzindo Audio

Vamos aprender a reproduzir audio no html5
Tag para inserir audio: audio src="audio/Ringside - Dyalla.mp3" controls autoplay

Tipos de audios extras: MP3 / WAV / OGG
outra maneira de colocar o audio:
audio preload="metadata" controls loop
• source src="audio/Ringside - Dyalla.mp3" type="audio/mpeg" (recomendado)
• source src="audio/Ringside - Dyalla.ogg" type="audio/ogg" (recomendado)
• source src="audio/Ringside - Dyalla.wav" type="audio/wav"
